

# About the TAPP Template

The TAPP template is a starter template for [Tauri Apps](https://tauri.app) and desktop companions.

Being an opinionated framework, it installs and sets upp the following high-level building blocks.

- [VueJS](https://vuejs.org) as frontend framework.
- [Vue Router](https://router.vuejs.org/) for routing on the frontend.
- [Pinia](https://pinia.vuejs.org/) for _reactive_ state management.
- [VueUse](https://vueuse.org/) for a great baseline of composable utilities.
